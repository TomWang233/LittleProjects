# This is the SQL project

## Two tables in database are used, custom_info2 and fund_detail.

## Fields in custom_info2:

mobile

branch_num

deposit: in CNY

mobile_bank: 1-using mobile bank

## Fields in fund_detail:

Event_ID

Party_ID

ZONE_NUM

FUND_CD
